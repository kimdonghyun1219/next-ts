#####################################
####### Message Structure ###########

# type: Subject

# body

# footer

#####################################

##### The Type #####
# feat: 새로운 기능 추가
# fix: 버그 수정
# docs: 문서 교체
# style: 스타일, 코드컨벤션  수정 (로직 변경이 없는)
# refactor: 성능 최적화 및 코드 리펙토링
# test: 테스트 코드 작성 및 수정
# chore: 패키지 매니저 수정, 빌드 수정

##### Subject #####
### The title does not exceed 50 characters
### and the first letter is capital letters.

##### Body #####
### Why?
### How?

##### Footer #####
### issue tracker id


##### Example #####
# feat: Create Notification message (#12345)
